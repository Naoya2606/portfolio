# event-manager

イベント運営に必要な情報を一元管理する、主催者向けのイベントマネジメントWebアプリです。

音楽イベントの主催・運営に携わる中で、出演者やスケジュールの情報が複数の連絡手段に分散し、連絡ミスや確認漏れが起きやすいという課題を感じ、それを解決するために制作しました。

## 概要

イベントの開催情報・出演者・会場・タイムテーブル・チケット状況などをまとめて管理できます。主催者は出演者情報やスケジュールを登録・確認でき、関係者との情報共有を効率化できます。運営に必要な情報を一か所に集約することで、連絡ミスや確認漏れを防ぐことを目的としています。

## 主な機能

- イベントの開催情報の登録・管理
- 出演者情報の登録・一覧表示
- 会場情報の管理
- タイムテーブルの作成・確認
- チケット状況の管理

## 使用技術

- フレームワーク: Next.js (App Router)
- 言語: TypeScript
- ORM: Prisma
- データベース: PostgreSQL
- スタイリング: Tailwind CSS
- ホスティング: Vercel

## 開発について

要件の整理、機能の検討、実装方針の決定を自分で行いながら、実装には生成AIを活用して開発を進めました。自身のイベント運営経験から「どんな情報を、どう管理できると運営が楽になるか」を考え、その要件をもとに機能を設計しています。

## デプロイ

Vercel でデプロイしています。
https://event-manager-snowy-one.vercel.app

## ローカルでの実行

```bash
npm install
npm run dev
```

`.env` にデータベース接続情報（`DATABASE_URL`）の設定が必要です。

ブラウザで http://localhost:3000 を開くと確認できます。
